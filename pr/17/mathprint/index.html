TI-84 Plus MathPrint renderer

Type an expression and it is typeset in 2-D using the calculator's real ROM font glyphs, following the page-0x39 MathPrint layout engine documented in the equation-display wiki page. This is a reconstruction of the layout model, not the emulator; the glyph bitmaps are extracted from the ROM by tools/export-font.py.

step the emission order

This steps the calculator's real draw order: after a measure pass the OS emits whole glyphs and rules one at a time at pen coordinates (0x86D7/0x86D8), in emission order — large glyphs via _PutMap (07:4588), small ones via _VPutMap (01:6293), the fraction rule via eqdisp_draw_fraction_bar (39:6abf). Each step highlights the element drawn and the row below names the routine. Stretched structural marks (the ∫ stem, parentheses, vinculum) appear at the final step.

Syntax
  • a/b — fraction (numerator over denominator)
  • a^b — superscript / exponent (raised 4 px)
  • sqrt(x) — radical with a vinculum sized to the radicand
  • int(lo,hi,body,var) — definite integral; the ∫ and its parentheses grow to the body height
  • abs(x), parentheses, + - * ^, and implicit multiplication (2X)

Geometry constants are the ROM-confirmed ones: 5 px large-font cells, a 7 px fraction column unit (x = base + 7·col), a row_height + 2 px row step, and a 4 px exponent raise.
